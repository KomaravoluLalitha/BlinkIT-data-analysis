# Blinkit Data Analysis Project

## Project Overview

This project focuses on performing **end-to-end data analysis** on **Blinkit grocery sales data** using multiple tools: **Excel, PostgreSQL, Python, and Power BI**.
The objective is to extract **business insights**, analyze **sales performance**, understand **customer behavior**, and visualize key metrics to support data-driven decision making.

---

##  Objectives

* Analyze Blinkit grocery sales data
* Identify sales trends and top-performing products
* Understand outlet-wise and item-wise performance
* Perform data cleaning and transformation
* Create interactive dashboards and reports
* Practice real-world data analysis using multiple tools

---

##  Dataset Description

The dataset contains Blinkit grocery-related data including:

* Item details
* Outlet size and location
* Sales figures
* Item type and category
* Ratings and visibility

This dataset is used to analyze **overall business performance** and **customer purchasing patterns**.

##  Analysis Performed

###  Excel Analysis

* Data cleaning and formatting
* Pivot tables for sales analysis
* Outlet-wise and item-wise performance
* Basic KPIs and summaries

##  How to Use This Project

1. **Excel**
   * Open `Blinkit_data_analysis_in_excel.xlsx`

2. **SQL**
   * Import dataset into PostgreSQL
   * Run queries from `BlinkIT_data_analysis_in_postgresql.sql`

3. **Python**
   * Open `Blinkit_data_analysis_Using_Python.ipynb`
   * Run cells sequentially

4. **Power BI**
   * Open `BlinkIT_data_analysis_in_powerbi.pbix`
   * Explore interactive dashboard
